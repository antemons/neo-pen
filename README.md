# NEO PEN

This software downloads the digital ink from a Neo spartpen N2.

## Getting Started

### Setup

 0. (Optional: install virtual environment)
 
        virtualenv env -p python3
        source  env/bin/activate

 1. Install: 
    
        pip install .

### Usage

        neo-pen <path to pen Data folder> ./
